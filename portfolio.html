Mon portfolio

Écrit par Théo Le Gourrierec - Web | UI / UX

Descriptif

Ce projet consiste à créer un site portfolio afin d'avoir une présence en ligne.

Pour pouvoir arrivé à ce résultat, j'ai dû commencer par fait différentes recherches graphiques pour trouver ce qui me représente. Ensuite, j'ai commencé à dessiner les wireframes sur papier ce qui a permit de délimiter chaque partie du site. Une fois les wireframes faites, j'ai attaqué la maquette sur Figma et commencé l'intégration en HTML / CSS et avec un peu de JavaScript pour classer les projets par types.

J'ai également cherché à avoir les meilleurs scores de perfomance, d'accessibilité et de SEO afin d'être le mieux référencé parmis tous les sites existants. Sur Google, quand on recherche "Théo Le Gourrierec" dans le moteur de recherche, c'est mon portfolio qui ressort en première place. Pour réaliser cela, j'ai du utiliser Google Search Console. J'utilise cet outil en complément de CloudFlare pour analysé les visites sur mon site.

Grâce à ce projet, j'ai pu créer mon propre portfolio afin de pouvoir exprimé les différents éléments qui me représentent et de quoi je suis capable.

Pas besoin de l'illustrer, il vous suffit de retourner à l'accueil pour découvrir le site !

Languages utilisés : HTML - CSS - JavaScript
