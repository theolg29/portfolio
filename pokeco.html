Pokéco - Gestion de pokémon

Écrit par Théo Le Gourrierec - Web

Descriptif

Pokéco est un site que j'ai créé pour un projet en MMI, le but était de pouvoir gérer des Pokémon, c'est-à-dire en ajouter, en supprimer et pouvoir consulté leurs descriptifs respectifs.

Pour réaliser ce site, j'ai dû utiliser HTML pour structurer le site, CSS pour le mettre en forme ainsi que PHP pour gérer le contenu affiché avec différentes fonctionnent qui agissent sur le code et modifie la page en direct.

Qu'es que j'ai pu apprendre durant ce projet ? Eh bien, je dirai que j'ai pu approfondir mes connaissances en HTML et CSS et de débuter en PHP, langage que je n'avais pas encore utilisé auparavant.

Languages utilisés : HTML - CSS - PHP
